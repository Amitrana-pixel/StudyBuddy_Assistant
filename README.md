# 📚 StudyBuddy Assistant

StudyBuddy Assistant is an AI-powered PDF chatbot built using **LangChain, Mistral AI, ChromaDB, and Streamlit**.

It allows users to upload PDF documents and interact with them conversationally using Retrieval-Augmented Generation (RAG).

---

# 🚀 Features

✅ Upload Multiple PDF Files  
✅ AI-Powered Question Answering  
✅ Retrieval-Augmented Generation (RAG)  
✅ Semantic Search using Embeddings  
✅ Context-Aware Responses  
✅ Source Chunk Retrieval  
✅ Beautiful Modern Streamlit UI  
✅ ChromaDB Vector Database Integration  

---

# 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- Mistral AI
- ChromaDB
- PyPDF
- Vector Embeddings
